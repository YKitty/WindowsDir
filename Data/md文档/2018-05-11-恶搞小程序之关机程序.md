# 恶搞小程序

---



这是一个**关机程序**，你可以在你同学的电脑上试一下。

他必须输入“我是猪”才可以让电脑停止关机

赶快来试试吧！嘻嘻



现在你有兴趣了吗？赶快来看看代码是如何实现的吧!

```c
#include<stdio.h>
#include<stdlib.h>
#include<string.h>

int main()
{
	char input[10] = { 0 };
	system("shutdown -s -t 60");
	while (1)
	{
		printf("电脑将在一分钟内关机，如果输入：我是猪，就取消关机!\n请输入：");
		scanf("%s", input);
		if (0 == strcmp(input, "我是猪"))
		{
			system("shutdown -a");
			break;
		}
	}
	return 0;
}
```

看着代码简单吧！只需要几行代码，赶紧快实现一下吧
